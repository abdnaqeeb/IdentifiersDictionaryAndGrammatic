A variable can be defined in different types:
1. Primitive type
	- [[number]]
	- [[string]]
	- [[boolean]]

### Scopes

A variable can be defined in different scopes:
1. Global
2. Class object field
3. Class object property
4. Function local
5. Function argument
6. Function parameter
7. Loop index
8. Loop iterator
9. Loop local
